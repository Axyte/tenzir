Transformations

VAST comes with the following transformation operators, in alphabetical order:

📄️ drop

Drops fields from the input. The dual to select.

📄️ extend

Appends fields to events.

📄️ hash

Computes a SHA256 hash digest of a given field.

📄️ head

Limits the input to the first N events.

📄️ measure

Replaces the input with incremental metrics describing the input.

📄️ parse

The parse operator converts raw bytes into events.

📄️ pass

Does nothing with the input.

📄️ print

The print operator converts raw bytes into events.

📄️ pseudonymize

Pseudonymizes fields according to a given method.

📄️ put

Returns new events that only contain a set of specified fields.

📄️ rename

Renames fields and types.

📄️ replace

Replaces the fields matching the given extractors with fixed values.

📄️ select

Selects fields from the input. The dual to drop.

📄️ sort

Sort events.

📄️ summarize

Groups events and applies aggregate functions on each group.

📄️ tail

Limits the input to the last N events.

📄️ taste

Limits the input to the first N events per unique schema.

📄️ unique

Removes adjacent duplicates.

📄️ where

Filters events according to an expression.
